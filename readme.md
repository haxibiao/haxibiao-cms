# haxibiao/cms

> CMS 能力

-   更新内容到站点
-   置顶内容到站点页面

> SEO 能力

-   更新内容自动提交百度 什么收录
-   自动更新站点地图
-   蜘蛛抓取记录监控
-   搜索来路流量监控
-   站点在百度等搜索平台的收录，索引量变化趋势

## Roadmap & TODOs

-   定时自动更新内容，提交收录...
-   蜘蛛流量报警
-   索引量，搜索流量变化报警

## 安装步骤

1. 执行`composer require haxibiao/cms`
2. 第一次安装, 执行`php artisan cms:install`,
3. 更新版本, 执行`php artisan cms:publish`,
